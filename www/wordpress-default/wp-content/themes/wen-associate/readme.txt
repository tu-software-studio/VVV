== Usage ==

=== Social Menu ===
1. In the admin section, go to “Appearance > Menus”
2. Click “create a new menu”
3. To add social link, click “Custom Links” and enter social link URL. Click “Add to Menu”
4. In “Theme Locations”, choose “Social Menu”
5. Click “Save Menu”

To show menu in header:
1. From “Customize” section, go to “Theme Options > Header” section.
2. Turn on “Show Social Icons” checkbox to show social icons.
3. Click on “Save & Publish” to save the changes.

=== Home Page Customization ===
1. There are four widget areas in home page.
  - Front Page : Main
  - Front Page : Lower Left
  - Front Page : Lower Right
  - Front Page : Bottom
2. Drag widgets in above widget area to enable custom home page. If no widgets are there, default blog listing is displayed in home page.

=== Footer Widgets ===
1. There are four footer widget areas.
  - Footer Widget 1
  - Footer Widget 2
  - Footer Widget 3
  - Footer Widget 4
2. Drag widgets in above widget area to show in footer widgets.
